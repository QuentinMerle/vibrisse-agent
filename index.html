Expert Knowledge

Is my code really private?
Yes. By default, Vibrisse uses Ollama to run models locally on your machine. No code or prompts are sent to any server unless you explicitly enable a cloud provider.
Which models work best?
For local use, we recommend Llama 3 (8B) or Mistral. If you have enough VRAM, Llama 3.3 (70B) is a powerhouse. Our setup wizard helps you choose the best tier for your hardware.
How does Web Search work?
Vibrisse integrates with Tavily. When enabled, the agent can fetch live documentation, API changes, or stackoverflow answers to supplement its local knowledge.
Does it support Vision?
Absolutely. You can drop screenshots or UI mockups into the chat. The agent uses multimodal models to analyze the design and help you implement it in code.
Can I extend it with MCP?
Yes. Vibrisse is MCP-native. You can connect standard MCP servers to give the agent access to your calendar, linear, github, or even custom local databases.
