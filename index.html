Questions & Answers

Is it compatible with Windows?
Yes. Vibrisse Agent is cross-platform. We provide a native PowerShell installer for Windows and a dedicated launcher for WSL2 environments.
How does the data stay private?
By default, all inference runs on your local Ollama server. Your API keys are encrypted with AES-256 and never leave your machine.
What is Ghost Mode?
Ghost Mode allows you to trigger the agent via comments like `// @vibrisse: add docstrings`. The agent detects the tag, performs the edit, and notifies you when it's done.
Can I use my own models?
Absolutely. Vibrisse Agent supports Ollama, vLLM, LM Studio, and any OpenAI-compatible local server.
